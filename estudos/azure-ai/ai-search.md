# Azure AI Search

## O que é
Serviço de indexação e busca com capacidades de enriquecimento e busca semântica.

## Principais conceitos
- Índices
- Enriquecimento de dados
- Embeddings
- Busca semântica

## Uso comum com Azure OpenAI
- Recuperação de informações
- Chat com documentos
- Q&A corporativo

Esse serviço é essencial para cenários de IA generativa corporativa.

# Azure OpenAI

## O que é
Serviço que permite uso de modelos generativos como GPT em ambiente Azure.

## Principais conceitos
- Prompts
- Tokens
- Temperature
- Limites de contexto

## Casos de uso
- Chatbots corporativos
- Assistentes internos
- Geração de texto e análise semântica

# Fundamentos de Inteligência Artificial – Roadmap

Este roadmap estabelece a base conceitual necessária para o uso
profissional de IA aplicada.

## Conteúdos abordados
- Conceitos fundamentais de IA
- Machine Learning e Deep Learning
- IA preditiva versus IA generativa
- Casos de uso e não uso de IA
- Noções de ética e responsabilidade em IA

Este roadmap serve como base para certificações e estudos avançados.
